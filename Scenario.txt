# Оплата заказа

### Цель: Произвести оплату за заказ.
### Пользователь: Клиент.

**Сценарий.**
+ Система выводит общую сумму заказа и предлагает клиенту ввести данные для оплаты заказа. 
+ Клиент вводит реквизиты карты. 
+ Система вызывает обмен данными с платежной системой для осуществления платежа.

**Исключения:**
+ Данные карты отклонены платежной системой.
+ Пользователь передумал и закрыл страницу.
